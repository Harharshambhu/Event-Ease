Module: Schedules — event timeline, session programming & logistics coordination
Lennd Operations-first: load-in/out, deliveries, performances Zoho Attendee-first: agenda builder, session tracks Cvent Enterprise-first: multi-track, room assignments, capacity
Phase 1 — Schedule structure & configuration (admin)
ElementsTasksObjectives
Event day definitionsLoad-in, show days, load-out, rehearsals Define the full event timeline including pre-event days (load-in, rehearsal, soundcheck) and post-event (load-out, teardown). Set start/end times per day. Lennd includes load-in/out as first-class schedule days. Zoho focuses on public-facing event days. Cvent supports multi-day with pre/post conference days Capture the full lifecycle — not just what attendees see, but the 3 days before and 1 day after that operations teams live through. Load-in scheduling is often more complex than show-day programming
Tracks / parallel streamsConcurrent programming lanes Create parallel tracks for simultaneous activities: Main stage + Breakout A + Breakout B + Workshop room. Each track runs on its own timeline. Zoho supports multi-track with filtering by track in the agenda view. Cvent supports unlimited tracks with room assignment per session. Lennd handles this via schedule categories (performances, deliveries, meetings) Model the reality of complex events where 5 things happen at once — the main keynote, a vendor setup in Hall B, a VIP lunch, a soundcheck on the second stage, and a production meeting, all at 2 PM
Venue / room mappingWhere each activity happens Assign physical locations to schedule slots: room names, stage names, outdoor areas, loading docks. Cvent integrates with Event Diagramming for floor plans and room capacity tracking. Zoho uses smart floor planning with venue maps. Lennd uses location fields within schedule items Every scheduled item has a "where" — so operations can produce room-by-room run sheets and attendees can navigate to the right place
Session types / categoriesKeynote, workshop, panel, break, logistics Define session types with distinct visual treatment: keynote (full-width, highlighted), breakout (standard), workshop (interactive icon), break (greyed), logistics-only (hidden from attendee view). Zoho supports session branding with logos and colors per type. Cvent supports custom session categories with registration-level visibility Differentiate between what attendees see (keynotes, panels, breaks) and what operations teams see (vendor load-in, AV setup, security briefing) — same timeline, different audiences
Time slot templatesRecurring schedule patterns Create reusable time blocks: "morning keynote slot" (9:00-10:30), "breakout block" (11:00-12:00), "lunch break" (12:00-13:00). Apply templates across multiple days to maintain consistent rhythm. Cvent supports session templates. Zoho agenda builder has drag-and-drop slot creation Avoid rebuilding the same schedule structure for each day — template it once, customize content per day
Phase 2 — Session & activity population (admin + content team)
ElementsTasksObjectives
Session creationIndividual agenda items Create sessions with: title, description, start/end time, track assignment, room/location, session type, capacity limit. Assign speakers (linked from speaker module). Zoho uses a visual agenda builder with drag-and-drop. Cvent allows sessions to be auto-created from accepted abstract submissions. Lennd creates schedule activities assignable to groups/people Build the programmatic content of the event — every talk, workshop, panel, and performance has a structured record with time, place, and people attached
Speaker-to-session linkingWho presents where and when Assign one or more speakers to each session. Speaker's bio, photo, and social links appear on the session detail. Zoho speakers are featured on the event website with their sessions auto-linked. Cvent links accepted abstract submitters directly to their session. Lennd assigns activities to people records from the CRM — appears on their portal profile Single source of truth for "who is speaking where" — no mismatch between the printed program, the website, and what the speaker thinks their slot is
Logistics activitiesNon-public operational items Schedule operations-only activities: vendor load-in windows, AV setup, security walkthroughs, catering delivery, rehearsals, equipment pickup/return. These appear on internal/vendor schedules but not on the public attendee agenda. Lennd excels here — activities appear on assigned groups' portal views. Cvent supports internal-only sessions. Zoho is weaker on operational scheduling The attendee sees "Keynote at 9 AM." The AV vendor sees "Stage setup 6-8 AM, soundcheck 8-8:45 AM, keynote starts 9 AM." Same timeline, role-filtered views
Conflict detectionOverlapping assignments Flag when a speaker is double-booked, a room is over-capacity, or a vendor has overlapping load-in times. Cvent has room conflict detection built into session scheduling. Zoho shows time conflicts in the agenda builder. Lennd relies on manual review via schedule view — no automated conflict detection Catch scheduling errors before they become onsite emergencies — a speaker can't present in two rooms at the same time, a room can't host two sessions simultaneously
Accepted-to-scheduled pipelineFrom speaker acceptance to agenda slot Accepted speaker proposals automatically populate as schedulable sessions. Admin drags them into time slots. Cvent has the strongest pipeline here: Abstract Management → accepted → session created → slot assigned → published to agenda. Zoho is building abstract management (coming 2026). Lennd handles this via manual activity creation linked to speaker groups Eliminate re-keying — speaker's proposal title, description, bio, and topic become the session record without anyone retyping it
Phase 3 — Publishing & distribution (multi-audience)
ElementsTasksObjectives
Public agenda / event websiteAttendee-facing schedule Publish the finalized schedule to the event website and mobile app. Attendees browse by day, track, topic, or speaker. Bookmark/favorite sessions for a personal agenda. Zoho auto-publishes to the event website with filter-by-track and bookmark functionality in the mobile app. Cvent publishes to the Attendee Hub with session registration and personalized agenda. Lennd does not have a public-facing agenda website — schedule is portal-internal Give attendees a polished, filterable, bookmarkable agenda that sells the event and helps them plan their day
Stakeholder portal schedulesRole-filtered views per group Each stakeholder sees only their assigned activities in their portal. Vendor sees their load-in time and operational slots. Artist sees their soundcheck and performance. Volunteer sees their shift schedule. Lennd is strongest here — activities assigned to groups/people appear on their portal profile. Cvent uses the Speaker Resource Center for speakers' personal schedule. Zoho shows speakers their session details from their speaker console Every stakeholder answers "when am I needed and where?" by checking one portal — not by searching a 200-session public agenda for their 3 appearances
Internal operations run sheetMinute-by-minute for stage/production team Generate a detailed run-of-show view: every activity across all tracks, minute-by-minute, with assigned personnel, AV requirements, and transition notes. Export as PDF/Excel for print. Cvent supports detailed session exports. Lennd exports schedule data via CSV. Zoho provides agenda export for offline use The production manager's bible — a printed or tablet-displayed document showing exactly what happens at every minute across every stage, with who's responsible for each transition
Real-time schedule updatesLast-minute changes pushed live When a session time changes or a speaker cancels, update the schedule in one place and it propagates everywhere: website, mobile app, portal views, speaker notifications. Zoho web updates automatically flow to the mobile app. Cvent pushes updates to the Attendee Hub. Lennd updates portal views in real time One change, everywhere updated — no more "the printed program says 2 PM but the app says 3 PM" contradictions that confuse attendees and embarrass organizers
Session registration / RSVPAttendee capacity management Allow attendees to register for specific sessions to manage room capacity. Waitlists when sessions fill up. Cvent has full session-level registration with capacity enforcement and waitlists. Zoho supports session RSVPs via the mobile app with ticket-type gating. Lennd does not support attendee-facing session registration (operations-focused, not attendee-focused) Prevent the "200 people show up for a 50-seat workshop" problem — capacity-managed sessions with waitlists ensure no one arrives to a locked door
Phase 4 — Live execution & post-event analytics
ElementsTasksObjectives
Session analyticsAttendance, engagement, feedback Track who attended which session, engagement metrics (poll responses, Q&A participation), and post-session feedback ratings. Zoho provides detailed session analytics with speaker performance metrics. Cvent tracks session attendance via badge scanning and LeadCapture. Lennd does not track session-level attendance (focuses on credential/catering check-in) Data-driven programming for next year — "the AI workshop had 3x the engagement of the compliance panel" informs which topics to expand and which to cut
Speaker feedback / surveysPer-session attendee ratings Push post-session surveys to attendees who attended. Rate speaker quality, content relevance, and session format. Zoho supports feedback forms with session-level analytics. Cvent has integrated survey tools with results pushed back to speakers via the Speaker Resource Center. Lennd does not include session feedback tools Close the feedback loop — speakers see how they were rated, organizers see which sessions drove satisfaction, sponsors see which topics their audience cared about
Schedule performance reportWhat worked, what didn't Post-event report showing: session attendance vs. capacity, top-rated sessions, time-slot utilization, no-show rates per session, track popularity comparison. Cvent has the most comprehensive reporting here. Zoho provides event success metrics including turnout ratio and speaker feedback. Lennd provides schedule data exports but no built-in schedule analytics Transform the schedule from a planning artifact into a strategic asset — "the 4 PM slot had 40% drop-off, let's move the marquee session there next year to retain audience"
